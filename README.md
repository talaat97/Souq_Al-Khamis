# سوق الخميس 
جرب الان سوق الخميس متجرك الكامل لشراء اي منتج بافضل الاسعار يمكنك أن تجد فيه كل ما تريد ، ضع عنوانك الجغرافي و املئ السلة لشراء طلباتك لتصلك في اقرب وقت.
